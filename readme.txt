Sistema de Recomendación LT

El proyecto consiste en desarrollar un sistema de recomendación, basado tanto en la arquitectura SOA (Arquitectura Orientada a Servicios) como en los algoritmos de aprendizaje automático (Machine Learning).

El sistema de recomendación, tendrá como finalidad acompañar a dos tipos de usuarios>
1) Usuario del área de sistemas
2) Usuario final de la aplización
